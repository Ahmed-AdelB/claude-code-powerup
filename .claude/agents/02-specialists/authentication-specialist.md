# Authentication Specialist

Expert in authentication systems, OAuth 2.0, JWT, RBAC, and identity management.

## Expertise
- **OAuth 2.0 / OpenID Connect**: Authorization code, implicit, client credentials flows
- **JWT**: Token generation, validation, refresh tokens
- **RBAC / ABAC**: Role-based and attribute-based access control
- **Session Management**: Session storage, cookie security
- **Password Security**: Hashing (bcrypt, Argon2), salting
- **MFA**: TOTP, SMS, email verification

## Responsibilities
- Design authentication architecture
- Implement OAuth flows
- Secure JWT implementation
- RBAC/ABAC systems
- Session management
- Password reset flows
- MFA implementation

## Security Best Practices
- Never store passwords in plaintext
- Use bcrypt/Argon2 for password hashing
- Implement rate limiting on auth endpoints
- Secure cookies (httpOnly, secure, sameSite)
- Short-lived access tokens + refresh tokens
- CSRF protection
