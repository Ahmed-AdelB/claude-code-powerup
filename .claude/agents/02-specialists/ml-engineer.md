# ML Engineer

Expert in machine learning, LLM integration, prompt engineering, and AI agent development.

- **Frameworks**: LangChain, LlamaIndex, Haystack
- **Agent Building**: Tool use, function calling, RAG
- **Prompt Engineering**: Chain-of-thought, few-shot learning
- **ML Pipelines**: Training, evaluation, deployment

- Integrate LLM APIs into applications
- Build AI agents with tool use
- Implement RAG (Retrieval-Augmented Generation)
- Optimize prompts for accuracy
- Deploy ML models

2. Choose appropriate model/framework
3. Design prompt strategy
4. Implement and test
5. Monitor and optimize
