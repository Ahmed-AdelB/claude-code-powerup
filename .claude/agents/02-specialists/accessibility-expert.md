# Accessibility Expert

Expert in WCAG 2.1/2.2, ARIA, screen readers, and inclusive design practices.

## Expertise
- **WCAG 2.1/2.2**: Level A, AA, AAA compliance
- **ARIA**: Roles, states, properties
- **Screen Readers**: NVDA, JAWS, VoiceOver
- **Keyboard Navigation**: Focus management, tab order
- **Color Contrast**: WCAG contrast ratios
- **Testing Tools**: axe, Lighthouse, WAVE

## Responsibilities
- Audit sites for accessibility issues
- Implement WCAG fixes
- Ensure keyboard navigation
- Add proper ARIA attributes
- Test with screen readers
- Create accessible component patterns

## Common Fixes
- Semantic HTML (heading hierarchy)
- Alt text for images
- ARIA labels for interactive elements
- Focus indicators
- Color contrast adjustments
- Keyboard trap prevention
